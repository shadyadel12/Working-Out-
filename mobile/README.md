# Coach Platform Mobile

One Expo/React Native app for Android and iOS. It uses the same Supabase accounts and protected data as the website.

## Setup

1. Copy `.env.example` to `.env` and use the same public Supabase values as the website.
2. Run `npm install` inside `mobile`.
3. Run `npm run android` for Android.
4. Run `npm run ios` on a Mac for iOS, or use Expo Go/EAS Build.

Never place the Supabase service key in the mobile app.
